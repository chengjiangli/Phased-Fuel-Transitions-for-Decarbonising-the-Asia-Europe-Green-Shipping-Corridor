# Phased-Fuel-Transitions-for-Decarbonising-the-Asia-Europe-Green-Shipping-Corridor
This repository contains the supporting data and model code for the manuscript "Phased Fuel Transitions for Decarbonising the Asia-Europe Green Shipping Corridor". Contents: LCA input/output data, Vensim SD model files, and source data for all main and supplementary figures.
